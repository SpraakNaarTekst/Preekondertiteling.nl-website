Hoe werkt het?

Het systeem werkt in 3 stappen

1. De predikant spreekt zijn woorden in de microfoon.

2. De woorden worden naar een computer gestuurd, die achterin de kerk staat. De software op de computer is geleerd de stem van de predikant te herkennen. De gesproken woorden worden door de software zo goed mogelijk omgezet in leesbare tekst.

3. Die leesbare tekst wordt door middel van een draadloos netwerk in de kerk verstuurd. Tablets in de kerk kunnen dit signaal ontvangen en tonen de gesproken tekst op de tablet. Het venster van de tablet is geoptimaliseerd voor leesbaarheid en duidelijkheid zodat de preek goed te lezen is.

 

Server en server software

De server is een computer die achterin de kerk of in een aparte ruimte staat opgsteld. De computer wordt voor de dienst opgstart door simpelweg op de aan/uit knop te duwen. De computer logt automatisch in, start de Spraak naar Tekst software op, en begint te luisteren naar spraak om deze om te zetten naar tekst.

Het spraakherkennings systeem maakt gebruik van bestaande spraakherkenningssoftware die de spraak omzet naar tekst. Daaromheen is een eigen schil geschreven die via een draadloos netwerk de aangesloten tablets van de tekst kan voorzien. Nieuwe tekst wordt naar de tablets 'gepusht', wat inhoudt dat nieuwe tekst direct, zonder vertraging, op de tablets verschijnt. De software kan tot honderden tablets tegelijk bedienen, als dat nodig is.

 

Tablet en tablet software

Nadat de server computer is aangezet, kan een tablet worden gepakt. Met het aanklikken van een icoontje wordt de tablet software gestart. Deze maakt automatisch verbinding met de server computer en wacht tot er tekst beschikbaar is.

De tablet software is gemaakt door Spraak naar Tekst. Het maakt gebruik van een rustig kleurenpalet, een eenvoudige bediening en instelbare lettergrootte. De tekst wordt op de tablet getoond zodra deze vanaf de server computer verstuurd is. Na de preek kan door middel van een knop op de tablet de server computer op afstand afgesloten worden. Daarna kan de tablet teruggelegd worden, en kan men direct naar huis. Bij de volgende dienst wordt de Server computer opgestart en werkt het systeem weer!
